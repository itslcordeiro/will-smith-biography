# Will Smith Biography

Projeto desenvolvido em HTML, CSS e JavaScript apresentando a biografia, carreira, conquistas e curiosidades sobre o ator e músico Will Smith.

## 📖 Sobre o Projeto

Este projeto foi criado com o objetivo de desenvolver uma página web interativa e visualmente atrativa sobre a trajetória de Will Smith.

O site apresenta informações sobre:

- Biografia
- Carreira artística
- Premiações
- Curiosidades
- Linha do tempo
- Design responsivo

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 📂 Estrutura do Projeto

```bash
WillSmith-main/
│
├── index.html
├── README.md
│
├── css/
│   └── main.css
│
├── js/
│   └── main.js
│
└── img/
    └── imagens do projeto
````

## 🎨 Funcionalidades

* Interface moderna
* Layout responsivo
* Sessões informativas sobre Will Smith
* Animações e interatividade com JavaScript
* Organização de arquivos em pastas separadas

## ▶️ Como Executar o Projeto

1. Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/will-smith-biography.git
```

2. Abra a pasta do projeto.

3. Execute o arquivo `index.html` no navegador.
